## Quick Start

### Teacher Quick Start

1. **Register**: Create your teacher account
2. **Create a Class**: Set up your first biology class
3. **Add Students**: Import or manually add students
4. **Assign Assessment**: Create or assign a practice assessment
5. **View Progress**: Monitor student performance on standards

### Student Quick Start

1. **Receive Login**: Get credentials from your teacher
2. **Take Assessments**: Complete assigned assessments
3. **View Progress**: Check your mastery levels
4. **See Predictions**: View your predicted EOC score
5. **Focus Learning**: Review recommended study areas
